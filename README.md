# Customer Churn Analysis

## 📌 Overview

This project analyzes customer churn data from a telecom company to understand why customers leave and identify key factors influencing churn.

---

## 🎯 Objective

* Analyze customer behavior
* Identify churn patterns
* Provide insights to reduce customer loss

---

## 🛠️ Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib

---

## 📊 Steps Performed

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)

---

## 🔍 Key Insights

* Customers with month-to-month contracts have higher churn
* Customers with higher monthly charges are more likely to leave
* New customers (low tenure) churn more
* Fiber optic users show higher churn

---

## 💡 Conclusion

Customer churn is influenced by pricing, contract type, and customer tenure. Businesses should focus on improving early customer experience and offering better pricing strategies.
